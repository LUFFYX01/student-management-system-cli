# 🎓 Student Management System (CLI)

## 📌 Description

A Python + MySQL based CLI application to manage student records, marks, attendance, and reports.

## 🚀 Features

* Admin authentication (bcrypt)
* Student CRUD operations
* Marks management
* Attendance tracking
* Student report generation
* CSV export
* Logging system
* Analytics dashboard

## 🛠 Tech Stack

* Python
* MySQL
* mysql-connector-python
* bcrypt

## ▶️ How to Run

1. Install dependencies:
   pip install mysql-connector-python bcrypt

2. Setup MySQL database

3. Run:
   python main.py

## 📂 Project Structure

* main.py → entry point
* student.py → CRUD operations
* auth.py → authentication
* marks.py → marks module
* attendance.py → attendance module
* report.py → reports
* validation.py → input validation
* analytics.py → dashboard
